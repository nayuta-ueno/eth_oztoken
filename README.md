# eth_oztoken

ERC-20 example using OpenZeppelin.

## prepare

```console
$ git clone https://github.com/nayuta-ueno/eth_oztoken.git
$ cd eth_oztoken
$ npm i
```

And edit truffle-config.js for your environment.

## develop

```console
$ truffle develop
> migrate
```

```console
$ node mytest.js
```
